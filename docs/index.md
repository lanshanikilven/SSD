# SSD

在`Github`上找到一个`SSD`目标检测算法实现 - [lufficc/SSD](https://github.com/lufficc/SSD)，这个工程不仅完美的实现了`SSD`算法，而且整体结构清晰，具有高可扩展性。所以新建了一个仓库，一方面是学习`SSD`算法，另一方面是研究整个训练框架，以便于其他算法的实现

## 训练框架

整个训练框架可分为以下几个部分：

1. 配置
2. 日志
      1. 文本日志
      2. 可视化日志
3. 数据
4. 检查点
5. 优化器
      1. 梯度更新
      2. 学习率调度
